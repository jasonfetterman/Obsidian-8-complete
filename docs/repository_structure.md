# OBSIDIAN-8 Repository Structure

This document explains the directory structure used in the OBSIDIAN‑8 project.

---

## Root Directory

README.md – project entry point  
assets/ – images, diagrams, and media  
docs/ – human‑readable documentation

---

## Engineering Documentation

00_System_Definition – system engineering documents

01_Mechanical – chassis and mechanical design

02_Electrical – power systems and electrical architecture

03_Control_Architecture – motion control and kinematics

04_Autonomy – swarm and mission logic

05_Perception – sensor systems

08_Software – software architecture

09_Testing – validation and qualification testing

10_Lifecycle – maintenance and operational procedures

---

## Design and Manufacturing

CAD – SolidWorks models

12_STL_Production – printable parts
